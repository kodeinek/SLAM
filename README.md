Mobile robotics research platform based on an iRobot Roomba 560 equipped with a ZED 2i RGB‑D stereo camera and NVIDIA Jetson Nano. The project implements a custom SLAM pipeline with occupancy grid mapping, PnP RANSAC–based pose estimation fused with IMU data, and Theta* path planning for real‑time autonomous navigation in indoor environments. The robot streams RGB‑D data from Jetson Nano to a remote server, where SLAM and path planning are executed, and then receives motion commands over the Roomba Open Interface.
